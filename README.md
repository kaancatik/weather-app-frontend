# Weather Application For Beginners

Weatherapp is a simple weather forecast app, which uses some APIs to fetch 5 day / current forecast data (including pressure, rain possibilities, humidity, visibility, sunset / sunrise times and local time) from the OpenWeatherMap and to fetch places, cities, counties, coords etc. The main goal of this app is to be a sample of how to build an high quality web application that uses the Architecture components, ReactJS and NodeJS.
